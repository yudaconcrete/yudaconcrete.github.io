# 🏗️ X/Twitter 推广作品目录

## 目标
制作海外推广内容，在 X (Twitter) 上发布，引流到华龙网站 (yudahualong.com)，
吸引海外建筑承包商、基建公司来询盘。

## 内容策略（4个系列）
1. **🏭 工厂系列** — 展示生产、焊接、装配、发货流程
2. **🏗️ 产品系列** — HZS各型号亮点、参数、应用场景
3. **📋 项目案例系列** — 俄罗斯、中亚、非洲项目现场
4. **📚 知识系列** — 怎么选型、固定站vs移动站对比等

## 目录结构
```
推广作品/X/
├── images/          ← 素材图片（从华龙原站下载）
├── post-*.html       ← X帖卡片HTML（打开截图后发X）
├── post-*-tweet-text.txt  ← 推文纯文案
├── output/          ← 成品截图/成品图片
└── README.md        ← 本文件
```

## 使用方法
1. 打开 post-*.html 文件在浏览器中查看
2. 截图（建议1200×675px或直接全卡截图）
3. 复制 post-*-tweet-text.txt 的文案
4. 发到 X，带上图片和文案

## 发布建议
- 频率：每天1条，不要刷屏
- 最佳时间：海外B2B客户在线时段（北京时间晚8点-早8点对应欧美/非洲白天）
- Hashtag：每次带3-5个，不要堆砌
- CTA：确保总带上官网链接和WhatsApp
